Animate

Animate manages PrimeFlex CSS classes declaratively to during enter/leave animations on scroll or on page load.

Import
import {AnimateModule} from 'primeng/animate';
Getting Started

Animate uses PrimeFlex animations, however it can perform animations with custom CSS classes too. Takes two properties enterClass and leaveClass to simply add-remove a class and enter/leave during scroll or page load to manage elements animation if the element is in viewport.

Enter/Leave Animation

<img pAnimate enterClass="scalein" leaveClass="scaleout" [parent]="parent" src="assets/showcase/images/galleria/galleria1.jpg" alt="image 1" class="w-full h-30rem mb-5 border-round shadow-2 animation-duration-1000 animation-ease-out">
Properties
Name Type Default Description
enterClass string null Selector to define the CSS class for animation.
leaveClass string null Class to add when item begins to get hidden.
parent any null Specifies the parent container which animation take place.
Events

Directive has no events.

Dependencies

None.
